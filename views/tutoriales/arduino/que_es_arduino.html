¿Qué es Arduino?

Arduino es una plataforma de electrónica abierta usada para la creación de prototipos tecnológicos, basada en software y hardware libre, flexibles y fáciles de usar.

Arduino UNO


Cuando hablamos de Open Hardware, nos referimos a que todo lo necesario para que puedas fabricar tu propia versión de Arduino se encuentra disponible, desde los planos esquemáticos del circuito, hasta el programa de Bootloader, necesario para que un micro controlador pueda ser programado.

Esquemáticos del circuito de Arduino


Si hablamos de Open Software, Arduino posee un IDE para programar la tarjeta física, en un lenguaje de programación de alto nivel cercano a lenguaje humano, este IDE transforma el código de Arduino en un código Hexadecimal, entendible por los microcontroladores.
